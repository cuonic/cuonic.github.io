Mazeball

Comment jouer ?

Utilisez les flèches du clavier ou posez une main à 30 cm au dessus du LeapMotion, et penchez la pour faire pivoter le plateau de jeu.

Chaque joueur dispose de trois vies, si vous faites tomber la boule du plateau vous perdez une vie.

Commandes

  • f: Plein écran
  • r: Redemarrer

Le code

Ce jeu est écrit en Javascript (Three.js et Physi.js), HTML et CSS.
